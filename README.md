# Valdoria

Jeu RPG en développement.

## Structure du projet

```
assets/
  icons-16x16/       → Icônes RPG 16x16 (objets, armes, consommables...)
  pixel-crawler/     → Sprites personnages, mobs, environnement
  fantasyville/      → Tileset et décors Fantasyville
src/
  scenes/            → Scènes du jeu (menu, carte, combat...)
  entities/          → Entités (joueur, ennemis, PNJ...)
  ui/                → Interface utilisateur (HUD, menus...)
maps/                → Cartes et niveaux
docs/                → Documentation du projet
```
